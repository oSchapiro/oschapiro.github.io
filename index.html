HTML5 & css Normalize

HTML5 is becoming so popular in web development due to a multitude of reasons. It allows websites to be standardized, meaning that it is much easier for people to create, edit and maintain a site. It allows you to perform actions using just html that previously required plugins such as flash. We use normalize.css because as its name hints, it normalizes elements across most browsers ending in a more consistent experience.
